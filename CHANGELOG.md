# Changelog

All notable changes to this project will be documented in this file.

## [0.1.2] - 2025-11-26

### Added

- Expanded README with clearer, plain-language explanation of what the tool does and the deduplication problem it solves.
- Documented ideal use cases, including scenarios where users have already been using Bitwarden and want to migrate remaining Apple Passwords entries without creating duplicates.
- Added rationale sections covering why the tool exists, background on Apple Passwords → Bitwarden migration, and why a dedicated password manager can be a better long-term choice than relying solely on Apple Passwords.

### Changed

- Reorganized README structure to surface “what this tool does,” “when this tool is a good fit,” and usage instructions more prominently for new users.

## [0.1.1] - 2025-11-26

### Added

- Added basic user feedback print statements to the script for improved progress visibility during execution without changing core functionality.

## [0.1.0] - 2025-11-25

### Added

- Initial public release of the Apple Passwords to Bitwarden migration script.
- README with detailed usage instructions and security notes.
- requirements.txt for dependency management.
- MIT License for open source distribution.
- Versioning adopted with Semantic Versioning best practices.
